# **App Name**: Cue Vision

## Core Features:

- Table View: Simulate a bird's-eye view of the pool table.
- Ball Placement: Enable precise placement of the cue ball and object balls on the virtual table.
- Rail Selection: Provide options to select the number of rails for the shot (1-5).
- Aiming Mode: Toggle between ball-first and rail-first aiming methods.
- Pocket Selection: Allow users to select one of the six pockets as the target.
- Shot Prediction: Using physics models to realistically simulate ball behavior, the AI tool suggests precise aiming point, power percentage, and necessary cue ball spin for the shot.
- Image Analysis: Analyze a photo of a real pool table to replicate the ball positions in the app.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) evoking precision and clarity, like a focused shot.
- Background color: Dark gray (#26292E), provides contrast and reduces eye strain for extended use.
- Accent color: Cyan (#00FFFF), highlights key elements like the predicted path and suggested aim point.
- Clean, sans-serif fonts ensure readability of angles and numbers.
- Minimalist, geometric icons for shot settings.
- Fluid transitions between the photo-analyze mode and the simulator.
